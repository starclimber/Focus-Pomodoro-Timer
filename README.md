## 📖 项目简介 / Project Introduction
### 中文
这是一个纯前端实现的番茄工作法计时器，旨在帮助用户提升专注力和工作效率。项目完全基于原生 HTML、CSS 和 JavaScript 构建，无需任何第三方依赖，可直接在浏览器中运行。

**核心特性：**
- 🎯 **标准番茄工作法**：25分钟专注 + 5分钟短休，每4个番茄钟后进行一次长休息（15分钟）
- ⚙️ **高度可配置**：支持自定义专注时长、短休息时长、长休息时长及长休息触发周期
- 🔔 **多重提醒机制**：集成声音提示、桌面通知及屏幕内弹窗，确保不错过任何节点
- 📊 **本地数据统计**：自动记录当日完成的番茄钟数量和总专注时长
- 💾 **数据持久化**：使用 `localStorage` 保存设置与当日统计数据，刷新页面不丢失
- 📱 **响应式设计**：完美适配桌面端与移动端设备
### English
This is a pure front-end Pomodoro Timer designed to help users improve focus and work efficiency. Built entirely with vanilla HTML, CSS, and JavaScript, it requires no third-party dependencies and runs directly in the browser.

**Key Features:**
- 🎯 **Standard Pomodoro Technique**: 25-minute focus sessions + 5-minute short breaks, followed by a 15-minute long break after every 4 Pomodoros.
- ⚙️ **Highly Configurable**: Customize focus duration, short break duration, long break duration, and the cycle for long breaks.
- 🔔 **Multi-channel Alerts**: Integrates sound alerts, desktop notifications, and in-app pop-ups to ensure you never miss a transition.
- 📊 **Local Statistics**: Automatically tracks the number of completed Pomodoros and total focus time for the current day.
- 💾 **Data Persistence**: Uses `localStorage` to save settings and daily stats, ensuring data survives page refreshes.
- 📱 **Responsive Design**: Fully optimized for both desktop and mobile devices.
